# 前端项目模板

不同的分支代表不同的项目模板

| 分支名            | 说明                              |
| :---------------- | :-------------------------------- |
| vue3              | vue3 项目模板                     |
| vue3-element-plus | 基于 vue3 模板，集成 element-plus |
| vue3-vant         | 基于 vue3 模板，集成 vant         |
| vitepress-doc     | 基于 viteress 的文档模板          |
| vue3-ui-lib       | 用于编写 vue3 ui 组件             |
